# Requirement or Problem statement & (Thought Process) Solution Approach

## 1. Problem statement

- Write a program to print all even numbers from an array.

## 2. Understand the problem with sample inputs & outputs

### Sample - 1

- Input: [10, 3, 5, 2, 7, 6, 9]
- Output: 10, 2, 6

### Sample - 2

- Input: [12, 13, 14, 17]
- Output: 12, 14

## 3. Approach & solution notes

<details>
  <summary><b>Approach</b></summary>

- Use for loop to traverse / visit each element in the Array
- Add if condition to check Even Number & print it
  - if(number % 2 === 0) print(number)

- Make sure dry run with sample examples with notebooks

- Complexity
  - Time Complexity: O(n), where n is length of Array
  - Space Complexity: O(1)

</details>
